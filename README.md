## 📦 Sobre

**Unimake.DFe** é uma DLL desenvolvida para facilitar a integração de softwares ERP com os sistemas de documentos fiscais eletrônicos da SEFAZ. Suporta múltiplos tipos de documentos:

- NFe / NFCe
- MDFe / CTe
- NFSe / GNRE
- EFDReinf
- Entre outros
